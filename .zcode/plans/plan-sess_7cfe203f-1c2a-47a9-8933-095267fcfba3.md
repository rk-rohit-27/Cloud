
# Implementation Plan: Web Hosting Product Pages + Domains Page

## Overview
Add 5 new public pages (4 product landing pages + 1 domains page) to the existing ZypherHost site, in the Hostinger style. The existing homepage stays as-is; the Navbar and Footer get updated links to the new pages.

Each product page is a **full marketing page**: hero, feature grid, pricing table (3 tiers), FAQ accordion, CTA section.

---

## PART 1: Shared UI Components (new, reusable)

### 1.1 `src/components/public/PageHero.tsx` (NEW)
A reusable hero section for product pages. Props:
- `badge` (string) — small uppercase tag
- `title` (ReactNode, supports gradient spans)
- `subtitle` (string)
- `icon` (Lucide icon component)
- `primaryCta` / `primaryHref` / `secondaryCta` / `secondaryHref`
- Uses the same dot-grid + ambient glow orbs + framer-motion animations as the main Hero
- Centered layout, responsive

### 1.2 `src/components/public/PricingTable.tsx` (NEW)
Reusable pricing table with 3 tiers. Props:
- `plans` array: `{ name, tag, monthly, yearly, description, features[], highlighted }`
- Monthly/yearly toggle (reuses existing toggle styles)
- Same card design as the homepage Pricing component (glass-card, neon ring on highlighted plan)

### 1.3 `src/components/public/FeatureGrid.tsx` (NEW)
Reusable feature grid. Props:
- `features` array: `{ icon, title, description }`
- 2/3-column responsive grid with staggered scroll animations
- Same card styling as homepage Features component

### 1.4 `src/components/public/FAQ.tsx` (NEW)
Accessible accordion FAQ. Props:
- `items` array: `{ question, answer }`
- Collapsible rows with chevron rotation animation (framer-motion)
- One open at a time (or multiple — standard accordion pattern)

### 1.5 `src/components/public/CTASection.tsx` (NEW)
Reusable bottom CTA band with gradient background, headline, subtext, and two buttons.

### 1.6 `src/components/public/PublicLayout.tsx` (NEW)
Wraps Navbar + main + Footer for all public pages so each page doesn't repeat it.

## PART 2: Product Landing Pages (4 pages)

Each page composes the shared components above with hosting-specific copy and data. All are server components (no client state needed except the PricingTable's toggle, which is self-contained).

### 2.1 `/web-hosting` — `src/app/web-hosting/page.tsx` (NEW)
- **Icon**: `Globe`
- **Hero**: "Web Hosting that's fast, secure, and effortless"
- **Features**: Free SSL, LiteSpeed server, cPanel, 1-click WordPress, daily backups, free CDN, email accounts, 99.9% uptime
- **Plans**: Single ($2.99/mo), Premium ($4.99), Business ($8.99)
- **FAQ**: migration, money-back guarantee, domain inclusion, WordPress support

### 2.2 `/php-hosting` — `src/app/php-hosting/page.tsx` (NEW)
- **Icon**: `Code2` or `FileCode2`
- **Hero**: "PHP Hosting optimized for speed and compatibility"
- **Features**: PHP 5.6–8.3 selectable, MariaDB, Composer/SSH, LiteSpeed Cache, phpMyAdmin, Git deployment
- **Plans**: Starter ($3.49), Developer ($6.99), Agency ($14.99)
- **FAQ**: PHP version switching, SSH access, Composer support

### 2.3 `/cloud-hosting` — `src/app/cloud-hosting/page.tsx` (NEW)
- **Icon**: `Cloud`
- **Hero**: "Cloud Hosting with dedicated resources that scale"
- **Features**: Dedicated RAM/CPU, isolated cloud environment, instant scaling, NVMe storage, auto-failover, full root access
- **Plans**: Cloud Starter ($9.99), Cloud Pro ($24.99), Cloud Enterprise ($54.99)
- **FAQ**: scaling limits, isolation, dedicated IP

### 2.4 `/vps-hosting` — `src/app/vps-hosting/page.tsx` (NEW)
- **Icon**: `Server`
- **Hero**: "VPS Hosting with full root control and blazing NVMe speed"
- **Features**: Full root access, KVM virtualization, choice of OS (Ubuntu/Debian/AlmaLinux), IPv4+IPv6, SSD/NVMe, snapshot backups
- **Plans**: VPS 1 ($5.99), VPS 2 ($12.99), VPS 4 ($29.99)
- **FAQ**: OS choices, root access, upgrade path
